div.blueBorder: This selector targets div elements with the class "blueBorder." It will select div elements that have the class "blueBorder" directly applied to them, like div class="blueBorder">.../div>. It won't select elements inside a div> that has the class "blueBorder."
div .blueBorder: This selector targets elements with the class "blueBorder" that are descendants of a div>. It can select any element with the class "blueBorder" that is nested within a div>, not just div> elements. For example, it will select p class="blueBorder">.../p> if it is inside a div>, \in addition to any other element with the class "blueBorder" within a div>.
